# 주요 기능

## <span style="color:#417EDC">회원 (Member)</span>
1. **회원가입 및 로그인**
    - 사용자가 새로운 계정을 생성하고 시스템에 로그인합니다.

## <span style="color:#DE9F40">가게 (Store & Menu)</span>
1. **가게 탐색**
    - 사용자가 다양한 가게를 검색하고, 각 가게의 메뉴를 조회합니다.

2. **가게 및 메뉴 관리**
    - 가게 관리자가 메뉴를 추가, 수정, 삭제합니다.
    - 가게 관리자가 주문 내역을 조회하고, 주문 상태를 업데이트합니다.

## <span style="color:#64D483">주문 (Order & Delivery)</span>
1. **주문 생성**
    - 사용자가 가게에서 메뉴를 선택하고, 주문을 생성합니다.
    - 주문이 생성되면 결제 처리 요청이 자동으로 생성됩니다.

2. **주문 관리**
    - 가게 관리자가 주문 내역을 조회하고, 주문 상태를 업데이트합니다.

3. **주문 내역 조회**
    - 사용자가 과거 주문 내역을 조회하고, 상세 정보를 확인합니다.

4. **배송 요청 생성 및 처리**
    - 결제가 완료된 주문에 대해 배송 요청이 생성됩니다.
    - 배송 요청이 접수되면 배송 상태가 'PREPARING'에서 'WAITING', 'DELIVERING', 'DELIVERED'로 변경됩니다.

5. **자동 알림 및 업데이트**
    - 결제 상태가 변경되면 사용자에게 결제 완료 알림이 전송됩니다.
    - 배송 상태가 변경되면 사용자에게 배송 진행 상황 알림이 전송됩니다.

6. **주문 취소 및 환불 처리**
    - 사용자가 주문을 취소하면 환불 요청이 생성됩니다.
    - 환불 요청이 처리되면 결제 상태가 'REFUNDED'로 변경되고, 환불 내역이 기록됩니다.

## <span style="color:#DFE12C">결제 (Payment)</span>
1. **결제 처리**
    - 결제 처리 요청이 접수되고, 사용자가 결제를 완료합니다.
    - 결제가 완료되면 결제 상태가 'UNPAID'에서 'PAID'로 변경됩니다.
    - 결제가 완료되면 배송 요청이 자동으로 생성됩니다.

2. **결제 상태 변경 기록 확인**
    - 관리자가 결제 상태 변경 내역을 조회하여, 결제와 관련된 이력을 확인합니다.
